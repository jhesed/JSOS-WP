== Description ==

The "Embed Image Links" WordPress plugin can save you a lot of time, when creating posts or pages for your website.
Just paste a link to any online image directly in your text, and this plugin will automatically replace it with the image that it is pointing to.
Includes options to automatically scale images that are too large for the content area.

== Installation ==

1. Upload the folder 'embed-image-links' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start inserting links to images in your pages and posts - then publish them and watch the links change into images
